**Highlights**

- Implemented on-device AI assistant for privacy-first support
- Delivered 15+ secured REST APIs with JWT auth and real-time messaging

[`Repository`](https://github.com/Abhiram1106/SmartCare-Plus)

### 2) LinguaConnect (AI Language Platform)
**Stack:** React, TypeScript, Gemini, Whisper, OCR  
Offline-capable product to preserve endangered languages through speech, text, and image processing.

**Highlights**
- Designed PWA architecture for low-connectivity usage
- Integrated speech-to-text + OCR pipelines for multimodal input
- Built gamified modules to improve engagement and retention

[`Repository`](https://github.com/Abhiram1106/LinguaConnect)

### 3) Sentiment Analysis Platform
**Stack:** FastAPI, PyTorch, DistilBERT, RoBERTa  
Ensemble NLP system for high-accuracy sentiment classification at scale.

**Highlights**
- Achieved **98% model accuracy**
- Batch-processed thousands of CSV/XLSX records per run
- Exposed inference services through scalable API endpoints

### 4) EduVoyage (Admission Management System)

University admission workflow platform with AI recommendations and analytics export.

### 5) Web Navigator AI Agent

### 6) Real-Time Chat App (Android)
